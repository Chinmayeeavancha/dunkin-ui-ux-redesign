# 🍩 Donut & Coffee App UI Design

## 📌 Project Overview  
This project is a **Donut & Coffee Ordering Mobile App UI** designed to deliver a fast, visually appealing, and seamless food ordering experience.

The app allows users to browse menus, explore deals, add items to cart, and place orders effortlessly.

---

## 🎯 Problem Statement  
Users often face:
- Cluttered food ordering interfaces  
- Difficulty exploring menus quickly  
- Poor visual hierarchy  
- Complicated checkout processes  

This design solves these problems with a **clean, engaging, and user-friendly UI**.

---

## 🎯 Project Objectives  
- Create a smooth food ordering experience  
- Improve product discoverability  
- Design an intuitive cart & checkout flow  
- Enhance engagement with offers & deals  

---

## 👥 Target Users  
- Food delivery app users  
- Coffee & dessert lovers  
- Young mobile-first users

## 🔄 User Flow  
1. User lands on Home screen  
2. Browses menu or deals  
3. Adds items to cart  
4. Reviews cart  
5. Completes checkout  
6. Receives order confirmation  

---

## 🎨 UX Design Decisions  
- Bright and engaging color palette (orange & pink)  
- Card-based layout for clarity  
- Strong visual hierarchy  
- Clear call-to-action buttons  
- Minimal and intuitive navigation
